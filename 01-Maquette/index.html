Intégration web

Utiliser les langages HTML et CSS pour créer des pages web!

Langages Utilisé

  • Le HTML

    Le HTML permet de construire la structure de la page web, de déterminer les éléments qui composent notre page web

  • Le CSS

    Le CSS permet d'appliquer du style sur les éléments dréés dans le HTML afin de mettre en forme la page web.

  • Création de page web

    On construit des pages web en créant du code HTML et CSS qui est interprete par le navigateur web.
